4.3 AI Matching Algorithm
Match Score Calculation Process:
1. Skills Matching (`skillMatchingService`)
For each required skill:
- Check for exact match (100% weight)
- Check for partial match (50% weight)
- Check for related skill (25% weight)

Skills Match Score = (Matched Skills ГѓвЂ” 1.0 + Partial Matches ГѓвЂ” 0.5) / Total Required Skills ГѓвЂ” 100
2. Experience Matching
Experience Score = min(100, (Candidate Years / Required Years) ГѓвЂ” 100)
If candidate exceeds requirement: 100%
If candidate below requirement: Proportional score
3. Qualifications Matching
Check required qualifications against candidate:
- Exact match: 100%
- Equivalent qualification: 80%
- Related qualification: 60%
- Missing: 0%
4. Location Compatibility
If same location: 100%
If commutable distance: 80%
If remote-friendly role: 100%
If location mismatch: 0%
5. Salary Alignment
If candidate expectation within range: 100%
If slightly above range: 80%
If significantly above: 50%
If below range: 100% (advantageous)
6. Overall Match Score
Overall Score =
(Skills Match ГѓвЂ” 0.4) +
(Experience Match ГѓвЂ” 0.3) +
(Qualifications Match ГѓвЂ” 0.2) +
(Location ГѓвЂ” 0.05) +
(Salary Alignment ГѓвЂ” 0.05)
