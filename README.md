# UPhire IQ

AI-powered recruitment platform.

## Getting Started

The application lives in the `uphire/` directory. See **[uphire/README.md](uphire/README.md)** for:

- Local setup and environment configuration
- Running tests
- Deployment (Netlify)
- Architecture overview

```bash
cd uphire
npm install
npm run setup:env
# Edit .env with your Supabase credentials
npm run dev
```
